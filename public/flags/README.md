# Country Flag SVGs

This directory contains SVG flags for client testimonials.

## Current Flags

- `gb.svg` - United Kingdom
- `ng.svg` - Nigeria
- `es.svg` - Spain

## Adding More Flags

To add more country flags:

1. Create an SVG file named with the country's ISO code (lowercase)
2. Example: `ae.svg` for UAE, `us.svg` for USA
3. Use simple, optimized SVG code

## Usage

These flags are used in the `ClientReviews` component to show client countries.
