## üéØ Conversion Tracking Setup

### Conversion Actions to Track:

1. **Form Submissions** (Primary)
   - Value: $500
   - Category: Lead
   - Tracking: Thank-you page view

2. **Phone Calls**
   - Value: $300
   - Category: Lead
   - Tracking: Google forwarding number

3. **WhatsApp Clicks**
   - Value: $200
   - Category: Lead
   - Tracking: Click event

4. **Booking Widget Completions**
   - Value: $600
   - Category: Lead
   - Tracking: Booking confirmation

5. **Email Clicks**
   - Value: $100
   - Category: Lead
   - Tracking: Click event

### Implementation:

Add to `app/thank-you/page.tsx`:

```typescript
'use client';

import { useEffect } from 'react';

export default function ThankYou() {
  useEffect(() => {
    // Google Ads Conversion
    if (typeof window !== 'undefined' && window.gtag) {
      window.gtag('event', 'conversion', {
        'send_to': 'AW-XXXXXXXXX/XXXXXXXXXXXXX',
        'value': 500.0,
        'currency': 'AED'
      });
    }

    // Facebook Pixel
    if (typeof window !== 'undefined' && window.fbq) {
      window.fbq('track', 'Lead', {
        value: 500.0,
        currency: 'AED'
      });
    }
  }, []);

  return (
    // ... your thank you page content
  );
}
```

## üéì Best Practices

1. **Always use ad extensions** - Increases CTR by 10-15%
2. **Test multiple ad variations** - Minimum 3 ads per ad group
3. **Monitor search terms daily** - Add negatives aggressively
4. **Use exact match for high-intent keywords** - Better ROI
5. **Implement remarketing** - 3-5x higher conversion rate
6. **Mobile-first approach** - 70%+ traffic from mobile in UAE
7. **Local language support** - Consider Arabic ads for local market
8. **Seasonal adjustments** - Increase budget during peak seasons
9. **Competitor monitoring** - Use Auction Insights
10. **Quality over quantity** - Focus on qualified leads
